
# Pandas DataFrame Basic Tools

This project explains some basic and useful Pandas DataFrame tools that help us understand and explore data easily. It covers functions like `head()`, `shape`, `info()`, `tail()`, and `describe()` to view data, check rows and columns, understand data types, and get statistical insights for better data analysis.

Features

head() – Displays the first rows of the dataset for quick preview.

shape – Shows the total number of rows and columns.

info() – Provides information about data types, null values, and dataset structure.

tail() – Displays the last rows of the dataset.

describe() – Generates statistical information such as count, mean, minimum, and maximum values for numerical data.
